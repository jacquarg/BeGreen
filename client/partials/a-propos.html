Présentation du concept

BeGreen est une appplication qui vous permet de diminuer vos émissions de CO2 par différents moyens :

  • Indication de vos émissions de CO2

    • données basées sur vos propres émissions
    • catégorisation par sources de consommation
    • aspect Quantified Self

  • Jeu communautaire (bientôt disponible)

    • rend la plateforme attractive, ludique, et addictive même pour une personne non-adepte de l’écologie
    • confrontation de son émission de CO2 avec d’autres joueurs
    • pour augmenter son score, obligation de diminuer ses émissions de CO2 en suivant

  • Objectifs

    • possibilité de vous fixer des objectifs mensuels, parviendrez-vous à les atteindre ?
    • gain de places dans le classement et attribution de badges (bientôt disponible)
    • conseils personnalisés (bientôt disponible)

Données utilisateur

BeGreen mesure vos émissions de CO2 grâce à l'ensemble de vos données. Bien que ce prototype se concentre uniquement sur vos tickets de caisse, BeGreen analysera par la suite les données suivantes :

  • transport : véhicule (AXA), géolocalisation (Orange)
  • achats sur internet (CozyCloud, Privowny)
  • habitation : superficie du logement (AXA), énergies domestiques (AXA)

Base Carbone

Afin de vous fournir des données aussi précise que possible nous nous sommes appuyés sur la Base Carbone créée par l'ADEME : http://www.basecarbone.fr/

Croisée avec chacune de vos consommations Intermarché, cette Base Carbone vous apporte une valeur aussi détaillée que globale de vos émissions de CO2.
